# react-web
Boilerplate for serving a react frontend, with ACL, registration etc. Needs a backend systems connected via rest API.

## Setup

Add all the node modules needed
```javascript
yarn install
```

Run dev enviroment (will start webpack webserver on 8080)
```javascript
yarn start
```
